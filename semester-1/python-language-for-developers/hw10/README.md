# Задание

В предыдущем домашнем задании к теме «Работа с базами данных. FastAPI+SQLAlchemy+Alembic» вы разработали модель данных для хранения данных из CSV-файла.

Теперь ваша задача — доработать предыдущее домашнее задание, добавив новые функции для работы с базой данных и интеграции с FastAPI:
1. Добавьте методы для обновления и удаления данных в базе.
2. Подключите класс для работы с базой данных к FastAPI-сервису.
3. Добавьте эндпойнты для CRUD: создания (Create), чтения (Read), обновления (Update) и удаления (Delete) записей.
